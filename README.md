Comandos Git

git init: Inicia o git na pasta gerando o .git;

git status: mostra o branch atual e quantos commits foram feitos;

git branch -M {nome}: cria uma branch;

git remote add Origin {linkDoRepositorio}: adiciona a pasta no repositorio;

git add {nomeDoArquivo}: adiciona o arquivo ao repositorio;

git commit -M "feat: {o que voce fez}": adiciona as mudanças no repositorio;

git push origin main: lança o arquivo com o commit ja feito pra la
