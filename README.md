## Comandos Git

<ul> 
    <li> <strong> git init </strong> - Inicia o git na pasta gerando o .git</li>
    <li> <strong> git status: </strong> - mostra o branch atual e quantos commits foram feitos</li>
    <li> <strong> git branch -M {nome}: </strong> - renomeia a branch atual</li>
    <li> <strong> git remote add origin {linkDoRepositorio}: </strong> - adiciona a pasta no repositorio</li>
    <li> <strong> git add . : </strong> - adiciona/atualiza os arquivos</li>
    <li> <strong> git commit -m "feat: {o que voce fez}": </strong> - adiciona as mudanças no repositorio</li>
    <li> <strong> git push origin main: </strong> - lança o arquivo com o commit ja feito pra la</li>
    <li> <strong> git checkout -b feature/{nome da branch} : </strong> - cria uma branch e ja te coloca dentro dela</li>
    <li> <strong> git checkout {nome da branch}: </strong> - entra na branch</li>
    <li> <strong> git push origin {nome da branch}: </strong> - manda para a branch que criei os arquivos</li>
    <li> <strong> git pull origin {nome da branch}: </strong> - puxa o que eu modifiquei na nuvem</li>
    <li> <strong> git branch -l <strong> - lista as branchs do meu computador </li>
</ul>
